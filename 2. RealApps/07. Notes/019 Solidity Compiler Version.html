At the end of 2018 Solidity Compiler Version 0.5.0 was released.

It introduces some minor changes (approximately 5 in total) that are not compatible with the previous version of Solidity(0.4.x).

Examples used in these course have been created, tested and compiled using v.4.x. Fortunately there are just a couple of changes needed (1-2 keywords in just some contracts) in order to compile them using Solidity v0.5.x.

I've documented all important changes of Solidity v5.0.x here: https://www.udemy.com/master-ethereum-and-solidity-programming-with-real-world-apps/learn/v4/t/lecture/13137642

Furthermore, I've created a source code example for both v0.4.x and v0.5.x, where it was necessary. You may find them in course resource directory: https://drive.google.com/drive/folders/1x8u4ZBHeSL65Kn81DND0iR4IOmH30EfhÂ
